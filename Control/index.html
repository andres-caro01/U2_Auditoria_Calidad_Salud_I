Gráfico de Control

Es una herramienta visual que monitorea un proceso a lo largo del tiempo para determinar si está bajo control estadístico, detectando variaciones anormales (causas especiales) que indican problemas de calidad y requieren acción correctiva, a diferencia de la variación normal (causas comunes) del proceso. Permite identificar tendencias, cambios y puntos fuera de los límites predefinidos (línea central y límites superior/inferior) para asegurar la estabilidad y mejorar la calidad. Supervisa la variación del proceso a lo largo del tiempo, ayudando a los usuarios a identificar tendencias, cambios y cualquier proceso fuera de control que requiera atención. (Álvarez H. , 2024)

Componentes principales

  • Puntos de datos: Mediciones recopiladas a lo largo del tiempo.
  • Línea central: Representa el promedio del proceso.
  • Límites de control: Superior e inferior: Se calculan a partir de los datos y definen el rango de variación esperada.

¿Para qué sirve?

  • Supervisar procesos: Visualiza el comportamiento de un proceso continuamente.
  • Identificar causas de variación: Distingue entre variaciones aleatorias (comunes) y variaciones que indican un problema (especiales).
  • Tomar decisiones: Ayuda a saber cuándo intervenir para corregir o ajustar el proceso, evitando el sobre control.
  • Mejorar la calidad: Permite una respuesta rápida y acciones preventivas para reducir defectos y costos.

Tipos comunes:

  • Gráficos de variables: Para datos continuos (longitud, temperatura).
  • Gráficos de atributos: Para datos de conteo (número de defectos).
