Hoja de Recolección de Datos y Estratificación: Organizan la información.

Es una herramienta de calidad que combina un formulario estructurado (hoja de datos o verificación) para registrar datos de forma ordenada con una técnica (estratificación) para clasificarlos en subgrupos (estratos) y así analizar patrones, causas de problemas y tomar decisiones efectivas, pasando de datos generales a información específica y accionable, útil en procesos productivos y de mejora. (Álvarez H. , 2024)

    Hoja de Recolección de Datos (Hoja de verificación / Chequeo)

    Qué es: Un formulario predefinido para recopilar datos de manera sistemática y visual, registrando la frecuencia de eventos, defectos o causas.

    Para qué sirve: Organizar información sobre variables, defectos, tareas, o la ocurrencia de problemas, facilitando el análisis posterior.

    Cómo se diseña: Incluye campos para identificar el origen de los datos (fecha, máquina, turno, operador) y un formato para totalizar la información recolectada

    Estratificación

    Qué es: Técnica estadística que divide un conjunto de datos en grupos más pequeños y homogéneos (estratos) para identificar patrones que no son visibles en los datos generales.

    Para qué sirve: Separar y comparar datos de diferentes fuentes o categorías (ej. turnos, máquinas, proveedores) para entender qué factor influye más en un problema o proceso.

    Cómo se aplica: Se usa junto con la hoja de datos para analizar por separado los registros de cada estrato, llevando a una comprensión más profunda y decisiones más precisas.

    Analizar: Usar esta información para identificar causas raíz y mejorar el proceso, pasando de un "qué" general a un "por qué" específico.

    Aplicación Conjunta

    Diseñar la Hoja de Datos: Crear un formulario que incluya categorías para estratificar (ej. columnas para 'Día', 'Máquina', 'Tipo de Defecto').

    Recolectar: Llenar la hoja registrando los datos en las categorías correspondientes.

    Estratificar: Analizar los datos agrupados por cada categoría (estrato) para ver dónde ocurren más problemas (ej. más defectos en la Máquina A, Turno 2).

    Hoja de Recolección de Datos (Check Sheet)

    Es un formato prediseñado (físico o digital) para recopilar datos de forma estructurada y en tiempo real. Su objetivo es facilitar el registro de hechos de manera que puedan ser analizados fácilmente después. (Álvarez H. , 2024)

    Tipos comunes:

    De registro de defectos: Para contar cuántas veces ocurre un problema específico.

    De localización de defectos: Un diagrama del producto donde se marca dónde ocurren las fallas.

    De causas de defectos: Clasifica los problemas según la máquina, el operador o el turno.

    Utilidad: Evita depender de la memoria y permite identificar patrones de frecuencia.
