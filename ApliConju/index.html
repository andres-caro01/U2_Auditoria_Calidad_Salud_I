1 Diseñar la Hoja de Datos

Crear un formulario que incluya categorías para estratificar (ej. columnas para 'Día', 'Máquina', 'Tipo de Defecto').

2 Recolectar

Llenar la hoja registrando los datos en las categorías correspondientes.

3 Estratificar

Analizar los datos agrupados por cada categoría (estrato) para ver dónde ocurren más problemas (ej. más defectos en la Máquina A, Turno 2).

4 Analizar

Usar esta información para identificar causas raíz y mejorar el proceso, pasando de un "qué" general a un "por qué" específico.
